Below are the quotas and storage limits for HCL Connections Cloud:

## Communities

Community Default: 50GB
Upload limit files: 2GB

## Wikis

Attachment: 2GB

Total size: 20GB

## Blogs

Maximum Media Size: 250MB
Max Size: 4096MB

## Forums

Max Upload: 100MB

Forbidden file types: exe,bat,sh,oa
